# LHN-Task-Management-App

1. git clone

```
git clone https://github.com/rkapril/LHN-Task-Management-App.git
```

2. Build docker image

```
docker build -t task-management-app .
```

3. Run docker image

```
docker run -p 3000:3000 -d task-management-app
```
